# Shopping Cart Side Project

這是一個個人製作的購物車 (Shopping Cart) 專案，用於練習全端開發，並整合前端與後端技術。  

## 功能 (Features)
購物車:
- 新增商品到購物車 (Create)
- 查看購物車內容 (Read)
- 更新購物車中商品數量 (Update)
- 移除購物車商品 (Delete)

訂單:
- 新增訂單 (Create)
- 查看訂單商品/地址 (Read)
- 移除訂單 (Delete)

地址:
- 新增地址 (Create)
- 查看地址 (Read)
- 更新地址 (Update)
- 移除地址 (Delete)

> 簡單實現完整的 CRUD 功能，作為個人全端練習專案。

## 技術棧 (Tech Stack)
- **前端:** React + TypeScript
- **後端:** Spring Boot + Spring API
- **資料庫:** MySQL
- **資料存取:** Hibernate / JPA
- **測試:** Postman + Junit/Mockito + Spring Boot Test
